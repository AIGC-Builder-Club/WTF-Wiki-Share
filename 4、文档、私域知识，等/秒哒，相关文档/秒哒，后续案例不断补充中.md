# 我自己制作的《葫芦岛盗号王传说》

- 产品极简概述：
	- 初始v1版：
		- `做一个名为《葫芦岛盗号王传说》的 Web/H5 整活社交平台 MVP。它类似 X.com 的轻量信息流，用户可以注册登录、设置昵称头像、人设标签和口癖、发布帖子、评论、点赞。核心玩法是：如果用户超过 72 小时未登录，系统会在用户下次登录时判定账号被“AI 盗号王”接管，并根据用户历史帖子和人设标签生成若干条 AI 盗号期间发布的帖子和评论。用户回来后会进入一个短小好笑的“找回账号”流程，找回成功后看到“盗号战报”，展示 AI 发了多少内容、最离谱帖子、最像本人帖子、社交损害指数等。用户可以对 AI 发布的内容选择删除、保留、发布被盗声明或认领为本人观点。MVP 需要预置 NPC 用户和初始信息流，让单个用户也能玩完整闭环。所有盗号都是站内虚构机制，不连接真实 Steam、X 或任何第三方账号。`

- 关键决策记录：
	- 1、秒哒本来，写【Design设计】文稿时，（可能望文生义）选择了【`赛博东北风暗黑社交平台红色警告蒸汽朋克`】   的设计风格？   实际编写后观看下来，这种风格极丑。
		- 之后，我强行口喷了一通，要秒哒去修改风格
			- 第1次，我说【`目前你这个网站的网页有点太丑了。并且你这个主页都让人提不起任何想要去玩的这种愿望，包括这个 UI 设计，看上去都是那种很太单薄，然后太无趣的展示。整体上配色和设计我感觉也不是一个很好的配色。      请你好好地重新思考一下，然后重新设计一下，然后对于整体的一个前端 UI 进行一个美化。`】
				- 然后，出来后还是    整体风格没变、只在一些细节上    做的稍微精致一点：
					- ta自己说：  `完成了《葫芦岛盗号王传说》全站 UI 的深度美化重设计，从零建立了"赛博东北朋克 × CRT老网吧终端"视觉体系`。
			- 第2次，我说【`我不喜欢这个东北风，也不喜欢这个黑乎乎的界面。 这个本身和东北也没啥关系，这个梗和东北地名关系也不大。`】
				- 最后成品， ta自己说  `完成了《葫芦岛盗号王传说》全站 UI 的彻底重设计，放弃了之前的黑色赛博东北朋克主题，改为**怀旧像素游戏（NES/FC复古）**风格：暖米色明亮背景、硬边像素阴影、方块按钮压感动效、RPG对话框样式、ChillBitmap 像素字体`  。
					- 我个人感觉：    `好的，这个风格赞爆了 很好。`    。
		- 第2次，最终完美  适配了风格。
